・使い方

jupyter notebook上でソースコードを「shift+Enter」で実行する。 
次にウインドウが表示されるので、「input window」を左クリックで1マス更新する。右クリックで広く更新する。
「q」を押せばシステムは終了する。

jupyter notebook:version 5.74
numpy:version 1.154
